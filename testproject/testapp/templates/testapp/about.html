How it works

morph-client.js intercepts all clicks on local <a> tags and form submissions. It fetches the new page via fetch(), sends the X-Django-Morph: true header, and uses idiomorph to morph the DOM.

The middleware converts 301/302 redirects into 200 responses with an X-Morph-Redirect header so the client can update history.pushState.
